関数の定義

functionで関数を宣言する

関数を定義するにはfunction文を使います。 
    function hoge(arg) {
        return arg * arg;
    }
関数の呼び出し
()を付けることによって関数を実行することが出来ます。
※()を付けるか付けないかはとても重要になってきますので、よく覚えておいてください。 
    hoge(10);

練習エリアで関数を実行してみましょう。

無名関数

名前を付けない関数を無名関数と呼びます。名前を付けずに、関数を変数に代入することが出来ます。
この書き方は関数リテラルと呼ばれる式となります。
また、Javascriptの関数は特殊なオブジェクトで、データとしてどこにでも記述出来ます。

  • 一度しか使わない命名する必要性のない関数をシンプルに記述出来る
  • 変数に代入出来るので、関数の引数などに渡せる
    •     // 変数hogeに関数を代入
    var hoge = function(arg) {
        return arg * arg;
    };
    hoge(10);

        // 変数fooに関数を代入
    var foo = function(func, arg) {
        // 引数で受け取った関数funcを実行
        return func(arg) * arg;
    };
    // 関数hogeを引数に渡す
    foo(hoge, 10);
    即時関数

    関数を定義し即座に実行する関数を即時関数と言います。

  • 関数の中にコードを閉じ込めるので、グローバルスコープを汚染することなくコードを記述出来る
  • 一度しか処理されない初期化処理などに有効
    •     // functionを()で囲む
    (function(arg) {

        return arg * arg;

    // ()を後ろに付けて即実行
    })(10);

    Javascriptのスコープはグローバルスコープと関数スコープしかありません。
    グローバルを汚染しないことはとても重要です。

    クロージャ

    コードとスコープが対になったものをクロージャと呼びます。

    前提として関数は次の性質を持っています。
  • 関数の中に関数を定義出来る
  • 関数を返す関数を作れる
    •     // 外側の関数
    function outer() {
        // valの初期化
        var val = 0;

        // 内側の関数
        function inner() {
            val++;
            return val;
        }
        // 関数を返す
        return inner;
    }

    // fooにはinner()関数が代入される
    var foo = outer();
    // inner()を実行
    foo();

    foo()を実行してみてください。
    またfoo()を連続で実行するとどうなるか確認してください。

  • 状態を保持したプライベート変数を擬似的に定義出来る
    • ※グローバルスコープのfooがinner()への参照を保持していると考えられます。
    (ガーベージコレクションされない)

    クラス

    厳密にクラスというものは存在しません。

    コンストラクタ

    new演算子を使ってコンストラクタを実装できます。

        // Dateクラスのインスタンスを生成
    var d = new Date();
    d.getFullYear();

    コンストラクタは関数の中に記述します。
    this.***はメンバ変数のようなものです。

        function Rectangle(w, h) {
        // thisはRectangle関数自身を指す
        this.width  = w;
        this.height = h;
    }

    // インスタンスを生成
    var rect = new Rectangle(2, 4);
    rect.width;
    プロトタイプ
    面積を計算するメソッドを追加してみましょう。 
        function Rectangle(w, h) {
        this.width  = w;
        this.height = h;
        // area()というメソッドを追加
        this.area = function() {
            return this.width * this.height;
        }
    }

    var rect = new Rectangle(2, 4);
    var a = rect.area();

    area()メソッドを追加することが出来ましたが、問題があります。
    new演算子でオブジェクトを生成する度に、area()メソッドが再定義されてしまいます。
    (メモリの消費)

        function Rectangle(w, h) {
        this.width  = w;
        this.height = h;
    }

    Rectangle.prototype.area = function() {
        return this.width * this.height;
    }

    全てのインスタンスで共有するプロパティについては、プロトタイプオブジェクトに追加しましょう。
    プロトタイプオブジェクトは親クラスのようなものです。

    プライベートメンバ

    Javascriptにはprivateといったアクセス修飾子がありません。
    擬似的にプライベート変数を実装するには以下のようにします。 

        function Rectangle(w, h) {
        // 変数w,hはゲッターメソッドを介さないとアクセス出来ません
        this.getWidth  = function() {return w;}
        this.getHeight = function() {return h;}
    }

    var rect = new Rectangle(5, 10);
    rect.getWidth();

    デザインパターンの一例

    (function (root, factory) {

    root.circle = factory(root.jQuery);

}(this, function ($) {

    function Circle() {
        this.r = 0;
    }

    Circle.PI = 3.14159;

    Circle.prototype.setRadius = function(r) {
        this.r = r;
    }

    Circle.prototype.area = function() {
        return Circle.PI * this.r * this.r;
    }

    return new Circle();
}));

circle.setRadius(10);
circle.area();
