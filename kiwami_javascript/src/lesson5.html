関数の定義

functionで関数を宣言する

    function hoge(arg) {
        return arg * arg;
    }
関数の呼び出し
()を付けることによって関数を実行する 
    hoge(10);

無名関数

名前を付けない関数
  • 命名する必要性のない関数をシンプルに記述出来る
  • 関数の引数などに渡せる
    •     var hoge = function(arg) {
        return arg * arg;
    };
    hoge(10);

    var foo = function(func, arg) {
        return func(arg) * arg;
    };
    foo(hoge, 10);
    即時関数
  • 関数を定義し即座に実行出来る
  • グローバルスコープを汚染することなくコードを記述出来る
    •     (function(arg) {
        return arg * arg;
    })(10);

    クロージャー

    ※前提
  • 関数の中に関数を定義出来る
  • 関数を返す関数
    •     function outer() {
        var val = 0;

        function inner() {
            val++;
            return val;
        }
        return inner;
    }
    var foo = outer();
    foo();
    foo()を連続して実行してみてください
  • プライベート変数を擬似的に定義出来る(状態を保持)
