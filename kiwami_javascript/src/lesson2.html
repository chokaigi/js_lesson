$.each()メソッドによるループ処理の使い分け

$.each()メソッドには２種類の記述の仕方があります。
セレクタで取得した要素をループ処理する場合
$('.element').each(function(index, item) {

});
alert(index + ":" + item.id);を記述して例題 HTMLから
テーブルレコード(tr)の取得を確認


配列またはオブジェクトを処理する場合
$.each(data, function(index, item) {

});
alert(index + ":" + item);を記述してdataの中身を確認

※引数のindexは0から始まる番号、itemはその要素または値です。
