HTMLドキュメントに対応するのがDocumentオブジェクト

ドキュメントを表示するウィンドウに対応するのがWindowオブジェクト

Windowオブジェクト

グローバルオブジェクト
全てのグローバル変数はウィンドウのプロパティとして定義される 
    var answer = 42;
    window.answer = 42;
windowは省略出来る 
    window.alert();
    alert();

Documentオブジェクト

Windowオブジェクトのdocumentプロパティ
HTMLドキュメントの制御
プロパティ 説明
body 現在のdocumentのbody要素
bgColor 現在のdocumentの背景色
URL 現在のdocumentのURL
title 現在のdocumentのタイトル
createElement() 指定したタグ名で新しい要素を生成
getElementById() 特定のidを持つエレメントへのオブジェクト参照を返す
write() ドキュメントにテキストを書き込む
